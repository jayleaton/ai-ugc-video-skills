# Content Creators

Use this folder for reusable AI UGC creator documentation: creator profiles, voice rules, visual identity, prompt templates, and campaign fit notes.

Do not commit generated creator image assets, MP4s, product screenshots, or editor exports. Keep those in the project where the ad is being produced.

## Suggested Structure

```text
content-creators/
  creator-example/
    README.md
```

For each creator, document:

- Creator name and role.
- Audience fit and industries.
- Backstory and personality.
- Speaking style and common phrases.
- Wardrobe, grooming, and visual identity.
- Preferred environments and props.
- Reference image prompts.
- Usage rules and negative prompts.
- Notes for video generation tools.

## Asset Policy

Generated assets should live outside this repo or inside ignored local folders such as `assets/`, `generated/`, `outputs/`, or a project-specific campaign package.
