# UGC Script Writing

Script writing skill for production-ready UGC ads, AI creator prompts, hook selection, and chunked video generation.

## What It Does

- Selects a viral hook from `references/viral-hooks.md`.
- Writes conversational UGC voiceover.
- Supports mid-funnel, full-stack, and animated-infomercial formats.
- Creates chunked production prompts for AI video tools.
- Checks product reveal timing, natural language, soft CTA, and AI voiceover risks.

## Files

- `SKILL.md`: main skill instructions.
- `references/framework.md`: script frameworks and strategic guidance.
- `references/viral-hooks.md`: viral hook library.
- `references/chunked-production.md`: scene/chunk production rules.
- `references/captions-testing.md`: caption and testing notes.
- `agents/openai.yaml`: display metadata for OpenAI/Codex-style skill UIs.

## Use It When

Use this skill when you need UGC scripts that sound spoken rather than written, especially for TikTok, Reels, Shorts, creator ads, or AI avatar videos.
