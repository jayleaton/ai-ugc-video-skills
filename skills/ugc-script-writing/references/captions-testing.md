# UGC Captions and Testing

Use this reference when creating organic captions, first comments, test sets, or creative scaling plans.

## Caption Principles

- Lowercase by default.
- Use conversational tone.
- Use one soft emoji maximum, optional.
- Avoid hashtags unless prior platform tests show they help.
- Keep to 1-2 short sentences.
- Do not directly sell. Allude to the topic and let the video do the work.

## Caption Templates

Confessional or accidental discovery:

```text
nobody told me [truth]. wish I had figured this out years ago
```

Tested it:

```text
spent way too much money on [failed alternatives] before I figured out why none of it worked
```

Myth buster:

```text
if you've been [surface behavior] thinking it would [desired outcome], read this
```

Universal:

```text
genuinely did not know this until recently. if you [problem], this might explain why nothing's been working for you
```

## First Comment

Put the actual link in the first comment. Keep the main caption clean. The first comment can be slightly more direct:

```text
linking what i'm using in my bio if anyone wants to check it out
```

## Initial Test Set

Build 6-10 scripts before scaling spend. Cover at minimum:

- All five variation types
- 2-3 demographic profiles
- 2-3 hook frameworks
- Both mid-funnel and full-stack formats
- At least one mass-desire deep pain point

## What to Watch

- Watch time and completion rate, especially first 3 seconds
- Cost per click by creative angle
- Comment sentiment
- Whether viewers repeat the analogy or payoff line
- Demographic breakdown of who the algorithm serves each script to

## Kill Criteria

- Watch time drops in first 3 seconds: hook is not working.
- Viewers watch but do not click: payoff or CTA is weak.

## Scale Criteria

- Comments quote the analogy or payoff line back.
- Once a winner emerges, write hook variations before fatigue sets in.

## Fatigue Defense

Rotate demographics and settings across the campaign. The same script with a different creator and setting can extend a winning angle before the audience tires of it.
